The REST API

You must have an API key to use most REST API operations. Retrieve your API key from the users page.

  • GET api/check?text=[text to check] - checks the supplied text against the current terms
  • GET api/terms?apiKey=[apiKey] - get the current list of terms in XML
  • POST api/terms?apiKey=[apiKey]&terms=[terms] - adds the comma-separated list of terms to the database, enabling stemming for terms ending in *.
  • GET api/users?apiKey=[apiKey] - get the current list of users in XML
